# ChocoPy Compiler 

Compiler for [chocopy](https://chocopy.org/) written with python


## Status

- [x] **Scanner**
- [ ] **Parser** (In Progress)
- [ ] **Semantic Analysis** 
- [ ] **Code Generation**
- [ ] **Assembler&Linking**

## Installation & Testing

```bash
# Clone the repository
git clone https://github.com/Ridju/chocopy.git

# Install dev dependencies 
pip install -e ".[dev]"


# Run the tests
pytest 
```
